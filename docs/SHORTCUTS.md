# ⌨️ WheelChair Editor 快捷键参考

## 编辑操作

| 快捷键 | Windows/Linux | macOS | 功能 |
|--------|---------------|-------|------|
| 粗体 | `Ctrl+B` | `⌘B` | 切换粗体 |
| 斜体 | `Ctrl+I` | `⌘I` | 切换斜体 |
| 下划线 | `Ctrl+U` | `⌘U` | 切换下划线 |
| 撤销 | `Ctrl+Z` | `⌘Z` | 撤销 |
| 重做 | `Ctrl+Y` 或 `Ctrl+Shift+Z` | `⌘Y` 或 `⌘⇧Z` | 重做 |

## 插入

| 快捷键 | Windows/Linux | macOS | 功能 |
|--------|---------------|-------|------|
| 行内公式 | `Ctrl+M` | `⌘M` | 打开行内公式编辑器 |
| 块级公式 | `Ctrl+Shift+M` | `⌘⇧M` | 打开块级公式编辑器 |
| 链接 | `Ctrl+K` | `⌘K` | 插入/编辑链接 |

## 文档管理

| 快捷键 | Windows/Linux | macOS | 功能 |
|--------|---------------|-------|------|
| 新建文档 | `Ctrl+N` | `⌘N` | 创建新文档 |
| 保存 | `Ctrl+S` | `⌘S` | 手动保存 |
| 搜索 | `Ctrl+F` | `⌘F` | 搜索文档 |

## 公式编辑器

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Enter` | 确认插入公式 |
| `Esc` | 取消并关闭编辑器 |

## Markdown 输入

| 输入 | 结果 | 按键 |
|------|------|------|
| `# ` | 标题 1 | Space |
| `## ` | 标题 2 | Space |
| `- ` | 无序列表 | Space |
| `1. ` | 有序列表 | Space |
| `> ` | 引用块 | Space |
| `` ` `` | 行内代码 | Space |
| `---` | 水平分割线 | Enter |
| `$...$` | 行内公式 | Space |
| `$$...$$` | 块级公式 | Enter |

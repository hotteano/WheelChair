# ⚡ WheelChair Editor 快速入门

5 分钟上手 WheelChair 编辑器！

---

## 🚀 第一步：启动应用

```bash
cd apps/demo
pnpm install   # 首次使用
pnpm dev
```

浏览器访问 `http://localhost:5173`

---

## ✍️ 第二步：创建文档

点击左侧边栏的 **"+ 新建文档"** → 输入标题 → 开始写作

内容**自动保存**，无需手动保存。

---

## 🎯 第三步：尝试核心功能

### 试试 Markdown 输入

在编辑区输入：

```
## 我的第一个标题

这是一段文字，**粗体**，*斜体*。

- 列表项 1
- 列表项 2

> 这是一段引用
```

输入后按**空格**或**回车**，看看会发生什么！

### 试试数学公式

按 `Ctrl+M` 打开公式编辑器，输入：

```
E = mc^2
```

然后按 `Ctrl+Enter` 插入。

或者直接在编辑器输入 `$E=mc^2$` 然后按空格。

---

## ⌨️ 常用快捷键

| 按键 | 功能 |
|------|------|
| `Ctrl+B` | 粗体 |
| `Ctrl+I` | 斜体 |
| `Ctrl+M` | 插入公式 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+N` | 新建文档 |

---

## 📦 第四步：导出文档

完成写作后：

1. 点击顶部 💾 按钮
2. 选择导出格式（推荐 .wc 格式备份）
3. 下载文件

---

## 💡 提示

- 所有数据保存在浏览器本地，清除缓存会丢失
- 定期导出 .wc 文件备份
- 双击公式可以编辑
- 使用左侧搜索框快速查找文档

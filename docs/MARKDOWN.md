# 📝 WheelChair Markdown 指南

WheelChair 编辑器支持**即时 Markdown 转换**，让你在写作时享受 Markdown 的简洁，同时获得所见即所得的体验。

---

## 什么是即时转换？

输入 Markdown 语法后按 **空格** 或 **回车**，内容会自动转换为富文本格式。

```
输入: ## 标题
按空格后: 自动变成 H2 标题
```

---

## 支持的 Markdown 语法

### 1. 标题

| 输入 | 结果 | 级别 |
|------|------|------|
| `# ` | 标题 1 | H1 |
| `## ` | 标题 2 | H2 |
| `### ` | 标题 3 | H3 |
| `#### ` | 标题 4 | H4 |
| `##### ` | 标题 5 | H5 |
| `###### ` | 标题 6 | H6 |

**提示**: 输入 `#` 后必须跟空格才会触发转换。

### 2. 列表

#### 无序列表
输入 `- ` 或 `* ` 或 `+ ` 后按空格：

```markdown
- 项目 1
- 项目 2
- 项目 3
```

#### 有序列表
输入 `1. ` 或 `2. ` 等数字后按空格：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

#### 嵌套列表
按 Tab 缩进创建子列表：

```markdown
- 父项 1
  - 子项 1.1
  - 子项 1.2
- 父项 2
```

### 3. 引用块

输入 `> ` 后按空格：

```markdown
> 这是一段引用文字
> 可以有多行
```

### 4. 代码

#### 行内代码
输入反引号包裹的内容：

```markdown
使用 `console.log()` 输出内容
```

#### 代码块
输入三个反引号后按回车：

````markdown
```javascript
function hello() {
  console.log("Hello World");
}
```
````

### 5. 文本样式

| 输入 | 结果 | 说明 |
|------|------|------|
| `**粗体**` | **粗体** | 双星号 |
| `*斜体*` | *斜体* | 单星号 |
| `~~删除线~~` | ~~删除线~~ | 双波浪线 |
| `` `代码` `` | `代码` | 反引号 |

### 6. 水平分割线

输入三个连字符、星号或下划线后按回车：

```markdown
---

***

___
```

### 7. 链接

```markdown
[链接文字](https://example.com)
```

### 8. 图片

```markdown
![替代文字](https://example.com/image.png)
```

### 9. 数学公式

#### 行内公式
输入 `$...$` 后按空格：

```markdown
爱因斯坦质能方程 $E = mc^2$ 非常著名。
```

#### 块级公式
输入 `$$...$$` 后按回车：

```markdown
$$
\sum_{i=1}^{n} x_i = \frac{1}{n}\sum_{i=1}^{n} x_i
$$
```

## 导出 Markdown

你可以将文档导出为 Markdown 格式（.md）：

1. 点击顶部 💾 保存按钮
2. 选择 "Markdown (.md)" 格式
3. 下载文件

导出的 Markdown 文件可以在任何支持 Markdown 的编辑器中打开。

---

## 导入 Markdown

WheelChair 支持导入 Markdown 文件：

1. 点击 "文件" → "导入"
2. 选择 .md 文件
3. 文档会自动创建并打开

支持的 Markdown 扩展：
- 标准 Markdown
- GitHub Flavored Markdown (GFM)
- LaTeX 数学公式

## 快速开始

### 启动应用

```bash
# 进入演示应用目录
cd apps/demo

# 安装依赖（首次使用）
pnpm install

# 启动开发服务器
pnpm dev
```

打开浏览器访问 `http://localhost:5173` 即可使用编辑器。

### 创建第一篇文档

1. 点击左侧边栏的 **"+ 新建文档"** 按钮
2. 输入文档标题
3. 在编辑区开始写作
4. 内容会自动保存到浏览器本地存储

## 编辑功能

### 文本格式

| 功能 | 工具栏按钮 | Markdown | 快捷键 |
|------|-----------|----------|--------|
| **粗体** | **B** | `**文本**` | `Ctrl+B` |
| **斜体** | *I* | `*文本*` | `Ctrl+I` |
| **下划线** | <u>U</u> | - | `Ctrl+U` |
| **删除线** | ~~S~~ | `~~文本~~` | - |
| **行内代码** | `<>` | `` `代码` `` | - |

### 段落样式

| 样式 | 操作方式 | Markdown |
|------|---------|----------|
| **标题 1-3** | 工具栏 H1/H2/H3 | `## 标题` |
| **正文** | 工具栏 ¶ | - |
| **引用块** | 工具栏 " | `> 引用` |
| **无序列表** | 工具栏 • | `- 项目` |
| **有序列表** | 工具栏 1. | `1. 项目` |
| **代码块** | 工具栏 `</>` | ` ```代码 ``` ` |
| **水平分割线** | - | `---` |

### 插入媒体

#### 图片
1. 点击工具栏 🖼 按钮
2. 输入图片 URL 地址
3. 图片将插入到光标位置

支持的图片格式：PNG、JPG、GIF、SVG、WebP

#### 链接
1. 选中文本或定位光标
2. 点击工具栏 🔗 按钮
3. 输入链接地址
4. 按 Enter 确认

## Markdown 支持

WheelChair 支持 **即时 Markdown 转换**，输入特定字符后按空格或回车即可自动转换。

### 输入规则

| 输入 | 结果 | 说明 |
|------|------|------|
| `# ` | 标题 1 | 1-6 个 `#` 对应 H1-H6 |
| `- ` | 无序列表 | 也可用 `* ` 或 `+ ` |
| `1. ` | 有序列表 | 数字加英文句点 |
| `> ` | 引用块 | 大于号加空格 |
| `` ` `` | 行内代码 | 反引号包裹 |
| ```` ``` ```` | 代码块 | 三个反引号 |
| `---` | 水平分割线 | 三个连字符 |

### Markdown 示例

```markdown
# 一级标题
## 二级标题
### 三级标题

这是**粗体**，这是*斜体*，这是~~删除线~~。

- 无序列表项 1
- 无序列表项 2
  - 嵌套列表项

1. 有序列表项 1
2. 有序列表项 2

> 这是一段引用文字

行内代码：`console.log('Hello')`

代码块：
```javascript
function hello() {
  console.log('World');
}
```

---
```

## 数学公式

### 输入方式

#### 1. 快捷键插入
- **行内公式**：`Ctrl+M`
- **块级公式**：`Ctrl+Shift+M`

#### 2. Markdown 语法
- **行内公式**：`$E=mc^2$`
- **块级公式**：`$$\sum_{i=1}^n x_i$$`

#### 3. 工具栏按钮
- 点击 Σ 插入行内公式
- 点击 ∫ 插入块级公式

### 公式编辑器

双击公式或点击确认后，会打开公式编辑器：

1. **编辑标签**：输入 LaTeX 代码
2. **常用符号标签**：点击快速插入常用符号
3. **实时预览**：下方实时显示渲染效果
4. **确认**：`Ctrl+Enter` 或点击确认按钮

### 常用 LaTeX 符号

| 符号 | LaTeX | 说明 |
|------|-------|------|
| $x^2$ | `x^2` | 上标 |
| $x_i$ | `x_i` | 下标 |
| $\frac{a}{b}$ | `\frac{a}{b}` | 分数 |
| $\sqrt{x}$ | `\sqrt{x}` | 平方根 |
| $\sum$ | `\sum` | 求和 |
| $\int$ | `\int` | 积分 |
| $\alpha$ | `\alpha` | 希腊字母 α |
| $\beta$ | `\beta` | 希腊字母 β |
| $\pi$ | `\pi` | 圆周率 |
| $\infty$ | `\infty` | 无穷大 |

### 公式示例

```latex
% 行内公式
$E = mc^2$

$\alpha + \beta = \gamma$

% 块级公式
$$\sum_{i=1}^{n} x_i = \frac{1}{n}\sum_{i=1}^{n} x_i$$

$$\int_{a}^{b} f(x) dx = F(b) - F(a)$$

$$\lim_{x \to \infty} \frac{1}{x} = 0$$
```

## 文档管理

### 创建文档

1. 点击左侧边栏 **"+ 新建文档"** 按钮
2. 或按 `Ctrl+N`
3. 输入文档标题（默认为"未命名文档"）

### 切换文档

- 在左侧边栏点击文档标题
- 当前活动文档会高亮显示

### 搜索文档

1. 在左侧边栏顶部的搜索框输入关键词
2. 文档列表会实时过滤显示结果
3. 支持模糊搜索（匹配标题）

### 重命名文档

1. 在文档列表中右键点击文档
2. 选择"重命名"
3. 输入新标题

### 删除文档

1. 在文档列表中点击文档右侧的 🗑 图标
2. 或右键点击文档选择"删除"
3. 确认删除（删除后无法恢复）

### 文档数据

- 所有文档自动保存在浏览器 **localStorage** 中
- 数据不会上传到服务器，完全本地存储
- 清除浏览器数据会导致文档丢失，请定期导出备份

## 导入与导出

### 导出文档

支持多种格式导出：

| 格式 | 说明 | 用途 |
|------|------|------|
| **Markdown (.md)** | 纯文本 Markdown | 通用格式，兼容性好 |
| **HTML (.html)** | 带样式的 HTML | 网页发布 |
| **JSON (.json)** | 编辑器内部格式 | 数据备份 |
| **Text (.txt)** | 纯文本 | 简单文本 |
| **WC (.wc)** | WheelChair 专有格式 | 完整备份（推荐） |

导出步骤：
1. 点击顶部 💾 保存按钮
2. 或选择"文件"→"导出"
3. 选择导出格式
4. 下载文件到本地

### 导入文档

支持的导入格式：
- Markdown (.md)
- HTML (.html)
- Text (.txt)
- WheelChair (.wc)

导入步骤：
1. 点击"文件"→"导入"
2. 选择本地文件
3. 文档将自动创建并打开

## 常见问题

### Q: 文档数据存储在哪里？

**A:** 文档存储在浏览器的 localStorage 中，完全本地存储，不会上传到任何服务器。清除浏览器数据会导致文档丢失，请定期导出 .wc 格式备份。

### Q: 如何在不同设备间同步文档？

**A:** 目前 WheelChair 是纯本地应用，不支持自动云同步。您可以：
1. 导出 .wc 格式文件
2. 通过云盘、邮件等方式传输文件
3. 在另一设备上导入文件

### Q: 数学公式显示不正确？

**A:** 请检查：
- LaTeX 语法是否正确
- 特殊字符是否已转义
- 尝试使用公式编辑器的"常用符号"标签插入模板

### Q: 能否多人协作编辑？

**A:** 当前版本（1.0.1）不支持实时协作。这是一个单用户本地编辑器。协作功能计划在后续版本开发。

### Q: 支持哪些浏览器？

**A:** 推荐使用以下现代浏览器：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

Internet Explorer 不支持。

### Q: 如何更换主题？

**A:** 点击顶部导航栏的 🌙/☀️ 按钮即可在亮色/暗色主题间切换。

### Q: 编辑器崩溃了怎么办？

**A:** 如果编辑器崩溃或白屏：
1. 尝试刷新页面
2. 清除浏览器缓存
3. 检查是否有过大的图片或文件
4. 如问题持续，请在 GitHub Issues 反馈

## ✨ 三大核心特色

### 1️⃣ Markdown 即时转换

无需记忆复杂的快捷键，像打字一样自然地写作：

```
输入 ## 标题    →  自动变成 H2 标题
输入 - 列表    →  自动变成无序列表
输入 $E=mc^2$  →  自动渲染为数学公式
```

**支持**: 标题、列表、引用、代码、公式等 20+ 种 Markdown 语法

---

### 2️⃣ 优雅的数学公式

内置 KaTeX 引擎，让学术写作更轻松：

- **行内公式**: 按 `Ctrl+M` 快速插入 `$E=mc^2$`
- **块级公式**: 按 `Ctrl+Shift+M` 插入复杂公式
- **可视化编辑**: 双击公式即可编辑，实时预览
- **常用符号**: 内置 18+ 常用 LaTeX 模板

```latex
$$\sum_{i=1}^{n} x_i = \frac{1}{n}\sum_{i=1}^{n} x_i$$
```

---

### 3️⃣ 极简的文档管理

告别繁琐的文件操作，专注创作：

- 📄 **自动保存**: 内容实时保存到本地，无需手动保存
- 🔍 **快速搜索**: 一键查找所有文档
- 📤 **多种导出**: Markdown、HTML、PDF、专有 .wc 格式
- 🌙 **主题切换**: 亮色/暗色模式，护眼写作

## 🚀 快速开始

### 5 分钟上手

1. **创建文档**: 点击左侧「+ 新建文档」
2. **尝试 Markdown**: 输入 `## 我的标题` 后按空格
3. **插入公式**: 按 `Ctrl+M` 输入 `E=mc^2`
4. **导出文档**: 点击顶部 💾 按钮选择格式

### 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+B` | 粗体 |
| `Ctrl+M` | 插入公式 |
| `Ctrl+N` | 新建文档 |
| `Ctrl+Enter` | 确认公式 |

## 💡 使用提示

### 在代码中使用快捷键

```typescript
import { KeymapManager, defaultKeymap, formatKeybinding } from '@wheelchair/core';

// 创建快捷键管理器
const keymapManager = new KeymapManager({
  defaultKeymap,
});

// 格式化快捷键显示
const key = formatKeybinding('ctrl+b');
console.log(key); // 在 Windows: "Ctrl+B", 在 Mac: "⌘B"
```

### 自定义快捷键

```typescript
import { KeymapManager } from '@wheelchair/core';

const keymapManager = new KeymapManager();

// 注册自定义快捷键
keymapManager.register({
  id: 'custom.save',
  command: 'editor.save',
  key: 'ctrl+shift+s',
  preventDefault: true,
});
```

### 平台检测

```typescript
const keymapManager = new KeymapManager();

// 检测当前平台
if (keymapManager.isMac()) {
  console.log('当前是 Mac 平台');
}

// 获取平台特定的快捷键
const keybinding = keymapManager.getBinding('bold');
const macKey = keymapManager.getPlatformSpecificKey(keybinding);
```

| `↵` | Enter/Return |
